# DeltaLatent Dataset
[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.17904906.svg)](https://doi.org/10.5281/zenodo.17904906)

This data set contains satellite images from wave, river and tide dominated deltas. The satellite images where collected using the Google Earth Engine (GEE) platform. We used the Landsat 8 Surface Reflectance Tier 1 Collection 2, which provides 30 m multispectral resolution (with a 15m panchromatic band) and global, rediometrically calibrated surface reflectance imagery (Earth Resources Obseration and Science (EROS) Center, 2020). 


### Repository Structure
- rivers/     79  images of fluvial dominated deltas
- tides/      89 images of tide dominated deltas
- waves/      88 images of wave dominated deltas
